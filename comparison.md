# Student Registration Status Checker

## Purpose of the Project
The purpose of this project is to implement a registration logic that checks a student's workload across four different programming languages: Python, Java, JavaScript, and C.

## What the Project Demonstrates
It demonstrates how different programming paradigms (Object-Oriented, Procedural, etc.) handle user input, conditional branching, and data output.

## How to Run Each Version
- **Python:** `python python/main.py`
- **Java:** `javac java/Main.java && java -cp java Main`
- **JavaScript:** `node javascript/index.js`
- **C:** `gcc c/main.c -o program && ./program`
